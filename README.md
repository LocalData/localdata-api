SURVEY API
==========

To install dependencies

    $ npm install
    
To set environment variables:

    $ source ./sentenv_local.sh
    
Run with

    $ node web.js

To test the API, use:

    $ make test

Our previous test setup: 

    $ node testclient.js seedforms
    $ node testclient.js getform FORMID

(See the testclient.js code for more.)
